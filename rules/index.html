# Le Sphérier Le `Sphérier` est la compilation d'un ensembles de règles, les plus simples possibles, de manière à ce que leur texte explicatif tienne sur un support de la taille d'une carte à jouer. [Liens externes](rules/links) # Règles de base Les règles font utilisent des `mots-clés` ou à des `symboles` pour rester concises. Ces règles cadrent des phases de jeu, et principalement : - Le combat - Le hors-combat [Phases de jeu](rules/game-phases/) La majorité des règles ci-dessous sont relatives au combat. Lorsqu'il s'agit de hors combat, cela est précisé dans un encart dédié. **Mot-clé** Un mot-clé est un terme clair et précis, qui désigne une définition d'un point de règle. Un mot clé est défini soit : sur une carte,dans la section [Règles de base] dans la section [Mécaniques spéciales] Si un terme utilisé dans un point de règle n'est pas clair, il faut le définir comme un mot-clé, de préférence sur une carte dédiée, venant avec la carte . **Symbole** Un symbole est une représentation graphique d'un mot-clé. # Personnage [Personnage](rules/characters/Personnage) Représentation d'un joueur dans le jeu. # Mécaniques spéciales > Ici sont regroupées toutes les mécaniques trop complexes pour être détaillées sur une carte.
