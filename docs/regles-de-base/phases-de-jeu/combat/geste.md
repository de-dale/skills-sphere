# Geste
> synonyme : acte, geste
> **Coût**
> **Déclencheur**
> **Effet**

Le **Geste** est la plus petite unité permettant de découper le jeu.
Elle correspond à la séquence du combat au cours de laquelle il se passe quelque chose. 

Un **Geste**, associe :

- un Effet
- un Déclencheur
- un Coût

- une **[Réaction](https://trello.com/c/vcCvdkOI)**
